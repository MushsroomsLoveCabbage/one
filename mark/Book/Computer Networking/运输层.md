#### 可靠传输与非可靠传输

##### UDP(user Datagram protocol)

* 请求头8个字节（64bit）

  | 16bit            | 16bit            |
  | ---------------- | ---------------- |
  | 请求地址         | 目的地址         |
  | 数据长度         | 校验和           |
  | 数据（可有可无） | 数据（可有可无） |

##### TCP(Transform control protocol)

* 请求头20字节（）

| 32bit                                                        |
| ------------------------------------------------------------ |
| 请求地址 16bit， 目的地址 16bit                              |
| 序列号（数据已经发送的字节）SEQ                              |
| 确认号（已经收到的数据报）ACK                                |
| 首部长度4\|保留未用4\|CWR\|ECE\|URG\|ACK\|PSH\|RST\|SYN\|FIN\|接收窗口16 |
| 网络校验和\|紧急数据指针（16bit）                            |
| 选项                                                         |
| 数据                                                         |
