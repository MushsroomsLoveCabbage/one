***名词术语***

* 虚拟内存中使用Virtual Page 作为最小单位 P = 2^p
* 直接版

```sequence
CPU-->PM: PA(Phasical Address)
Note right of PM: find address -> get Data
PM-->CPU: Data
```

* 增加虚拟内存

```sequence
CPU --> MMU: VA
MMU--> PM: PA
Note right of PM: find address -> get Data
PM-->CPU: Data
```
